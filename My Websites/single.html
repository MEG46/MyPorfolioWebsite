Radio Simulator



Hello Dear Readers,

Today, I would like to introduce you to "Radio Simulator", a radio simulator created using the names and frequencies of radio channels in Istanbul. While this project does not provide a real music listening experience like a traditional radio, it creates a real radio-like feeling with features such as displaying the frequency and duration of the channel being listened to.

The project has three main options: searching for a channel using frequency, searching for a channel using its name, and listening to the radio. With these options, you can search for channels either by frequency number or by channel name, or you can test a simulation of a real radio listening experience.

Radio Simulator is an open-source project and can be further developed in the future. You can access the project's source code on GitHub.

Here's the link : Radio Simulator GitHub Link

If you enjoy listening to the radio in Istanbul and want to experience a real radio feeling, you should try Radio Simulator.
