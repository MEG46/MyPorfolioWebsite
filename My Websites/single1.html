Coffee Shop



Hello Dear Readers,

Today I would like to tell you about my latest project, Coffee Shop. Coffee Shop is a cafe simulation program that aims to provide users with a virtual shopping experience.

In this project, I took special care to avoid the mistakes and unnecessary code lines I made in my previous projects. Coffee Shop offers 45 different products in 4 different food and drink categories. In addition, users can place multiple orders and choose between cash or credit card payment options.

After completing the shopping, users can enter information such as their name and surname and select a payment method. Then, users can print out their receipts and review the products and prices they have purchased.

I developed the project using C++ and shared all the codes on GitHub. I am leaving the GitHub link below for those who want to examine the project in detail.

Here's the link : Coffee Shop GitHub Link

I hope the Coffee Shop project provides users with an enjoyable experience and can be used as a useful learning tool.
